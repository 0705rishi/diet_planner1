# Diet Planner

A full-stack web application to help users plan and track their diet.  
Built with **React + Vite** on the frontend and Node.js/Express for backend APIs.

---

## 🚀 Features
- Create and manage personalized diet plans
- Track daily meals and calories
- Responsive UI built with React + Vite
- Secure backend with Node.js/Express
- Environment variables managed via `.env` (not committed to GitHub)

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB (or your chosen DB)
- **Tools:** ESLint, Git
